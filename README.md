**Wanderlust** is a full-stack travel listing platform where users can browse, create, and manage vacation destinations from around the world. Built with **Node.js**, **Express.js**, **MongoDB Atlas**, and **EJS**, it includes secure authentication, image uploads, location-based listings using the **Grocoding API**, and full CRUD functionality.

## Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (cloud database)
- **Templating Engine:** EJS
- **Authentication:** Passport.js (local strategy)
- **File Uploads:** Multer + Cloudinary
- **Geocoding:** Grocoding API (for location to coordinates)
- **Frontend:** HTML, CSS, JavaScript
- **Others:** Flash messages, Method override, Validation, Error handling

---

## Features

- Add, edit, delete travel listings
- Upload images for each listing using Cloudinary
- User authentication and ownership-based authorization
- Location support using **Grocoding API**
- Full RESTful routes
- Server-side validation and security practices
- Responsive and clean UI
